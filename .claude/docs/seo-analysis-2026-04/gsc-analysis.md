## 4. Page Cannibalization

### Critical cannibalization issues:

**1. Brand query "gåri" — 13 URLs competing**
The query "gåri" triggers 13 different Gaari pages. While the homepage gets 193 clicks, the impressions are split across `/no`, `/no/about`, `/no/guide`, `/no/for-arrangorer`, `/no/denne-helgen`, `/no/i-dag`, `/no/i-kveld`, various event pages, and EN pages. This is Google testing which page to show. Not a crisis since the brand CTR is 95%, but the `/no/guide` page getting 191 impressions for the brand query with 0 clicks suggests Google is confused about what that page is.

**2. Easter queries — 3 pages competing**
- "påske bergen 2026": lands on `/no/easter-bergen` (413 imp)
- "hva skjer i bergen i påsken": `/no/easter-bergen` vs `/no/paske`
- "påske i bergen": `/no/paske` vs `/no/easter-bergen`
- "bergen i påsken": both pages appear

Having both `/no/easter-bergen` and `/no/paske` creates confusion. The slug `easter-bergen` on a Norwegian path is also odd — it mixes English and Norwegian.

**3. 17. mai queries — 3+ pages competing**
- `/no/17-mai` (collection page)
- `/no/17th-of-may-bergen` (another collection page)
- `/no/events/ja-vi-elsker-2026-...` (event page)
- `/en/17-mai`
- `/en/17th-of-may-bergen`

"17 mai bergen 2026" (71 imp) triggers 5 different pages. The two NO collection slugs (`17-mai` and `17th-of-may-bergen`) are clear cannibalization.

**4. EN vs NO pages for same event**
Queries like "eric idle bergen", "aries spears bergen", "electric wizard bergen", "dead by april bergen" all trigger both the `/no/events/...` and `/en/events/...` versions. This is standard hreflang behavior but the NO page should dominate for Norwegian queries and EN for English. Check hreflang implementation.

**5. "marinehagen amfi bergen"** — 8 event pages competing
This venue query triggers 8 different event pages that happen to be at Marinehagen Amfi. No venue landing page exists, so Google picks whichever event page it finds. A venue page would consolidate this.

**6. "aktiviteter i bergen"** — split between `/no/free-things-to-do-bergen` (pos 43) and `/no/guide` (pos 50)
Both pages are deep in results, but they're competing with each other rather than consolidating authority.

## 7. Actionable Recommendations

Ordered by estimated impact (impressions recoverable x realistic CTR improvement):

### HIGH IMPACT

**1. Fix title tags and meta descriptions for top event pages** (~3,000+ impressions affected)
- Event pages for "Ja vi elsker", "Barnas messe", "Vintoget", "Eric Idle", "Fjordsteam" have huge impressions but <2% CTR.
- Current meta descriptions are likely AI-generated summaries. Replace with structured snippets: "17. mai 2026, Festplassen Bergen. Bergens storste 17. mai-feiring med Kygo, Aurora og mer."
- Include date, venue, key artists in the title tag, not just the event name.
- **Estimated gain**: +50-100 clicks/period

**2. Consolidate Easter pages** (~2,200 impressions across pages)
- Merge `/no/easter-bergen` and `/no/paske` into a single canonical Easter page. Keep `/no/paske` as the primary (Norwegian slug on Norwegian path).
- Redirect `/no/easter-bergen` to `/no/paske`. Use `/en/easter-bergen` for the English version.
- The current Easter content ranks at pos 7.9 with 1,778 impressions — consolidating should push it higher.
- **Estimated gain**: +20-40 clicks/period

**3. Consolidate 17. mai pages** (~430 impressions across pages)
- `/no/17-mai` should be the primary Norwegian page. Redirect `/no/17th-of-may-bergen` to `/en/17th-of-may-bergen` (or drop it).
- Having two Norwegian slugs for the same topic splits link equity.
- **Estimated gain**: +10-20 clicks/period

**4. Improve /no/konserter for "konserter bergen" queries** (755 impressions, pos 14.7)
- This page has significant impressions but ranks on page 2. The title should be "Konserter i Bergen denne uken" (not just a collection label).
- Add editorial content with venue names (Grieghallen, Ole Bull, Forum Scene) for keyword relevance.
- The page also catches "konsert bergen idag", "live musikk bergen" — important navigational queries.
- **Estimated gain**: +15-30 clicks/period

**5. Create a "Messer i Bergen" collection page** (~1,000+ impressions capturable)
- Cluster: "barnas messe" (250), "friluftsmesse bergen" (210), "friluftsmesse grieghallen" (50), "barnemesse grieghallen" (14), "platemesse bergen" (14), "babymesse bergen" (25)

- This is the single biggest content gap by search volume.
- **Estimated gain**: +20-40 clicks/period

### MEDIUM IMPACT

**6. Improve /no/guide for "hva skjer i bergen" queries** (pos 37.3 for core query)
