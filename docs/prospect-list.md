# Prospect List — Gåri Venue Partnerships

Last updated: 2026-02-27

## Tiers

- **Partner** (7,000 NOK/mo) — 35% slot share, large venues/platforms
- **Standard** (3,500 NOK/mo) — 25% slot share, mid-size venues
- **Basis** (1,000 NOK/mo) — 15% slot share, smaller venues
- **A la carte** (500 NOK/event) — single-event promotion
